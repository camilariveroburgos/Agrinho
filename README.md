# AgroForte Quizz
Uma aplicação web interativa desenvolvida para testar e expandir os conhecimentos sobre a força, a importância e o impacto do agronegócio. Através de um jogo de perguntas e respostas dinâmico, o usuário pode aprender mais sobre o setor que movimenta a economia e a tecnologia no campo.
